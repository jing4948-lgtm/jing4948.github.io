# 다영의 웹페이지 📚

개인 프로젝트 모음 웹사이트입니다.

## 🌟 프로젝트 소개

이 웹사이트는 교육용 인터랙티브 웹 애플리케이션들을 모아놓은 포트폴리오입니다.

### 포함된 프로젝트

#### 1. 🤖 딥러닝 모험: AI 트레이너 게임
- **파일**: `deeplearning.html`
- **설명**: 딥러닝의 작동 원리를 게임으로 학습하는 교육용 프로그램
- **주요 기능**:
  - 3단계 학습 과정 (데이터 수집 → 학습 → 예측)
  - 신경망 시각화
  - 인터랙티브한 학습 경험
  - 고양이/강아지 분류 예제

#### 2. 🏫 학급 자리 배치 프로그램
- **파일**: `seatingarrangement.html`
- **설명**: 학급의 학생 자리를 무작위로 배치하는 웹 애플리케이션
- **주요 기능**:
  - 학생 수 설정 및 이름 자동 생성
  - 드래그 앤 드롭으로 책상 위치 변경
  - 무작위 자리 배정
  - 열 추가/삭제 기능
  - 애니메이션 효과

## 🚀 사용 방법

### GitHub Pages로 배포하기

1. 이 저장소를 GitHub에 업로드
2. Settings → Pages에서 배포 설정
3. `https://[username].github.io`로 접속

### 로컬에서 실행하기

```bash
# 저장소 클론
git clone https://github.com/jing4948/jing4948.github.io.git

# 디렉토리 이동
cd jing4948.github.io

# index.html을 브라우저에서 열기
```

## 🛠️ 기술 스택

- **HTML5**: 구조 및 마크업
- **CSS3**: 스타일링 (Glassmorphism, 애니메이션)
- **JavaScript**: 인터랙티브 기능
- **Tailwind CSS**: 유틸리티 우선 CSS 프레임워크

## 📁 파일 구조

